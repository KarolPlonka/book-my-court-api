API Features

🔑 Authorization

Registration.
Login.
Logout.
Email confirmation.
Password reset.

🎾 Courts & Venues

Create / Read / Update / Delete.
View information about courts and venues without authorization.
Retrieve all courts within a venue.
Retrieve information about reserved hours for a court.
Sort venues based on location.
Only users with "owner" privilege can create, edit, and delete courts and venues.
Privilege "owner" is granted only by the administrator.

📖 Reservations

Create / Read / Update / Delete.
Ability to assign a group to a reservation (if the user is a member of that group).
Prevent for creating reservation for an already reserved time slot.
Assigning group to a reservation automatically generates invitations for all of its members.
Retrieve information about responses to invitations from other users (poll).

👤 User

✉️ Group invitations

Read and respond to invitations to a group.
Read and respond to invitations to a reservation.
Retrieve the most frequently reserved courts for a user.

👥 Court Groups

Create / Read / Update / Delete.

✉️ Group invitations

Prevent invitations duplicates and inviting current members.
Only admins and the group creator can create invitations.

👥 Memberships

Automatic membership for the creator.
Delete members or edit their roles (admin, member).
Only admins and the group creator can edit roles and remove members.
Prevent removal of the group creator.
Adding a new member automatically generates invitations for future reservations assigned to the group.
Deleting a member deletes his invitations for future group events.

📖 Reservations

Delete reservation assignments for the group (the reservation still exists but is only assigned to the reservation creator).
Only admins and the group creator can delete reservation assignments not created by them.
